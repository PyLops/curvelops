Contributing documentation

Documentation is built locally witht the help of the Makefile in the root of the project. Use make doc to build the docs, make docupdate to build the docs without cleaning the build directories, and make watchdoc to rerun make docupdate at every change in files in curvelops/ or docssrc/source/.

Serve the documents locally with make servedocs.

Warning

PRs changing documentation should be submitted to the gh-pages branch.
